# Roadmap

## MVP

1. Finalize `ChartSpec` v1.1.0.
2. Add spec validation/migration CLI and CI gate.
3. Ship line/bar/area/scatter/pie/donut/combo/kpi renderers for iOS and Android.
4. Ship SwiftData and Room adapters with sample dashboards.
5. Add release/security automation for OSS operations.

## v0.2

1. Add combo chart support and annotations.
2. Add SQLDelight adapter and API pagination helpers.
3. Add performance benchmark harness.

## v1.0

1. Stable registry format and migration tooling.
2. Backward compatibility guarantees for spec minor versions.
3. Production-ready docs and starter templates.
