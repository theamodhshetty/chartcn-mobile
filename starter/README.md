# chartcn Starter Pack

User: mobile teams using SwiftUI + Compose dashboards.  
Promise: Ship production charts from ChartSpec in under few minutes.

This starter includes only 3 must-have templates:

1. `kpi-card`
2. `trend-line`
3. `comparison-bar`

Each template includes:

- `chartspec.json`
- `sample-data.json`
- `ios.swift`
- `android.kt`

One example spec for first-run review:

- `templates/kpi-card/chartspec.json`
